# 👨🏻‍💻 Pedro Rojas Gavidia — Portfolio

Personal portfolio of **Pedro Rojas Gavidia**, Senior Full Stack Engineer with 10+ years
of experience building and scaling web applications with Python, Django, TypeScript and
React for companies across the US and Europe.

Built with [Astro](https://astro.build/) and [Tailwind CSS](https://tailwindcss.com/).

## 🚀 Development

```bash
pnpm install   # install dependencies
pnpm dev       # start the dev server at http://localhost:4321
pnpm build     # type-check + production build
pnpm preview   # preview the production build locally
```

## 🗂️ Content

Most content is data-driven and lives in:

- `src/data/experience.json` — work experience timeline
- `src/data/projects.json` — featured projects
- `src/components/Hero.astro` / `src/components/AboutMe.astro` — bio and intro
